Eventos del Mouse con Bootstrap

Evento: click

Acción: Muestra un mensaje cuando se hace clic en el botón.

Evento: dblclick

Acción: Muestra un mensaje cuando se hace doble clic en el botón.

Mueve el mouse aquí

Evento: mousemove

Acción: Cambia el color de fondo cuando se mueve el mouse sobre el área.

Evento: mousedown

Acción: Cambia el color del botón cuando se presiona el botón del mouse.

Evento: mouseup

Acción: Cambia el color del botón cuando se suelta el botón del mouse.

Entra aquí

Evento: mouseenter

Acción: Cambia el texto cuando el mouse entra en el área del elemento.

Sale aquí

Evento: mouseleave

Acción: Cambia el texto cuando el mouse sale del área del elemento.

Evento: contextmenu

Acción: Evita que aparezca el menú contextual por defecto.

Rueda aquí

Evento: wheel

Acción: Muestra el desplazamiento de la rueda del mouse.

Posición aquí

Evento: mousemove

Acción: Muestra la posición del mouse dentro del elemento.
